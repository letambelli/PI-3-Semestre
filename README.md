# Projeto Integrador do 3º Semestre: LogiPlanner

Projeto desenvolvido no 3º semestre como parte da disciplina de Projeto Integrador: Analista de Desenvolvedor Web. O projeto consiste em um sistema de agendamento de entrega para uma transportadora.


## Objetivo

O projeto tem como objetivo criar um aplicativo web para o agendamento de serviços de transporte, visando aprimorar a gestão logística de uma transportadora.  

A aplicação foi desenvolvida para centralizar as informações de coletas e entregas, permitindo que um usuário admnistrador consulte, altere, exclua e crie novos agendamentos.


## Tecnologias Utilizadas

- HTML
- CSS
- JS
- PHP
- MySQL
